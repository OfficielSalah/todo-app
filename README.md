# Todo-App

# Introduction :

This Is a Web-App Built With Node.Js React.js And Mongodb Which Has Lot Of Features .

# Technologies Used :

- Node.js

# Structure :

- Sections:

  - Accounts .
  - todo :

    - completed .
    - active .
    - ASAP .
    - Danger Zone.

  - ideas .
  - Structuring Projects .
  - Roadmap .

# Features :

- More Documentation.
- Authentification .
- Authorization .
- Edit Notes .
- Drag Note From Section To Other .
- Set Up Deadline And Use Emails Or Messages or Both.
